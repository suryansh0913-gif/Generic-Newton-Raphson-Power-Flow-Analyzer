# Generic Newton-Raphson Power Flow Analyzer

## Description
A Python-based load flow analysis tool using the Newton-Raphson algorithm with PSS/E RAW file parsing and interactive Single Line Diagram (SLD) visualization.

## Features
- PSS/E RAW file parsing
- Y-Bus matrix generation
- Newton-Raphson load flow analysis
- Jacobian matrix computation
- Voltage profile analysis
- Interactive SLD visualization

## Technologies Used
- Python
- Pandas
- Tkinter
- PSS/E
